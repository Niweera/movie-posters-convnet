About

ConvNet-Posters is a project that aims to use the power of ConvNet to cluster similar movie posters.

You can use the the textbox as a search engine to find the movie you like. You can also click on any poster to get its own ''similar'' movie posters.

Another datavisualization is available here. It will plot 1,500 randomly picked movie posters into a 2D plot.
